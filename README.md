# BNK 부산은행 퇴직연금 로또볼 이벤트 랜딩페이지

퇴직연금 고객 행동(조회·진단·설정·신청)을 자연스럽게 유도하기 위해 제작된 **인터랙티브 웹 랜딩페이지**입니다.  
배경에는 **풀스크린 프로모션 영상**을 사용하며, 모든 UI는 **HTML/CSS/JS**로 구성되었습니다.

---

## 🔥 프로젝트 개요

본 랜딩페이지는 **부산은행 퇴직연금 고객을 대상으로 하는 로또볼 이벤트 참여 플랫폼**입니다.

사용자는 다음 행동을 수행할 때마다 **로또볼이 지급**되며,  
누적된 로또볼로 **경품 추첨에 자동 참여**하게 됩니다.

## 🎯 주요 기능

### 1) **배경 영상 (Full-Screen Video Background)**
- `bg.mp4` 기반 고해상도 영상 자동 재생
- `position: fixed`, `object-fit: cover` 적용으로 시각적 임팩트 극대화
- 콘텐츠는 반투명 오버레이 위에 배치하여 가독성 확보

---

### 2) **로또볼 획득 미션**
각 버튼 클릭 시 JS로 즉시 반영되는 데모 로직 포함  
(실제 운영 환경에서는 API 연동으로 대체)

| 미션 | 보상 |
|------|------|
| 퇴직연금 잔고 조회 | 1볼 |
| 운용현황 리포트 열람 | 1볼 |
| 투자성향 진단 완료 | 2볼 |
| 목표수익률 설정 | 2볼 |
| 디폴트옵션(책임투자배분) 신청 | 3볼 |

---

### 3) **나의 로또볼 실시간 업데이트**
- JS에서 `totalBalls` 상태를 직접 관리
- 증가 시 scale 애니메이션으로 시각적 피드백 제공

---

### 4) **반응형 UI**
- 모바일 퍼스트 기반
- 미션 카드 UI 구성
- 배경 영상 + 블러 오버레이 적용

# 📌 이벤트 상세 기획 (실제 서비스 수준)

## 1. 이벤트 목적
퇴직연금 고객의 핵심 행동(조회·진단·설정·신청)을 강화하기 위한 **행동 기반 리워드 캠페인**입니다.  
고객은 미션 수행 시 로또볼을 획득하며, 이벤트 종료 시 자동 추첨에 반영됩니다.

---

## 2. 사용자 흐름 (UX 플로우)

1) 랜딩 접속 → 이벤트 설명 확인  
2) 본인인증 → 보유 계좌 조회  
3) 고객 상태 기반 미션 목록 표시  
4) 미션 수행 → 완료 여부 자동 체크  
5) 로또볼 즉시 지급  
6) 마이페이지에서 누적 볼 확인  
7) 이벤트 종료 후 자동 추첨  
8) 당첨자 발표 및 알림톡 전송  

---

## 3. 미션 상세 정의 (개발 검증 기준 포함)

| 미션 | 행동 정의 | 검증 방식 | 지급 |
|------|-----------|-----------|------|
| 잔고 조회 | 연금 잔액 화면 접속 | `/balance/view` 로그 | 1볼 |
| 운용현황 리포트 열람 | 리포트 상세 진입 | 페이지 뷰 트래킹 | 1볼 |
| 투자성향 진단 완료 | 문항 완료 + 결과 저장 | DB 저장 이벤트 | 2볼 |
| 목표수익률 설정 | 목표값 저장 성공 | API 응답 성공 | 2볼 |
| 디폴트옵션 신청 | 제도 신청 완료 | 신청 플래그 변경 | 3볼 |

---

## 4. 로또볼 지급 로직

- 미션당 최초 1회만 지급  
- 반복 수행 시 중복 지급 불가  
- `user_mission_status`, `bonus_history`, `user_total_points` **3-테이블 구조 운영**  
- 지급 내역은 모두 히스토리로 저장하여 이력 검증 가능  

---

## 5. 추첨 방식

- **로또볼 1개 = 1회 응모권**  
- 이벤트 종료 후 자동 추첨 시스템 실행  
- 무작위 추첨 → 당첨자 선정  
- 앱푸시 / 알림톡으로 최종 결과 안내  

---

## 6. API 구조 (요약)

| Method | Endpoint | 설명 |
|--------|----------|------|
| POST | `/event/login` | 사용자 인증 |
| GET | `/event/missions` | 미션 상태 조회 |
| POST | `/event/missions/{missionCode}/complete` | 미션 완료 처리 |
| GET | `/event/points` | 누적 로또볼 조회 |
| GET | `/event/draw` | 관리자 추첨 실행 |

---

## 7. 향후 확장 아이디어

- **시즌제 운영** (분기별 로또볼 시즌)  
- **랜덤 데일리 미션** 제공  
- **부기 캐릭터 기반 아바타 성장 시스템**  
- **연금 교육 콘텐츠와 연동된 보상 구조**
